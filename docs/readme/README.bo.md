## 🌟 རྣམ་གཞག་ངོ་སྤྲོད

**bacs-bridge-server** ནི་ལིན་ཧྲུའུ་(飞书)འཕྲུལ་མི་དང་ AI CLI ལག་ཆ་(Claude Code / Codex)ཕན་ཚུན་ཟམ་པ་སྦྲེལ་བའི་མ་ལག་ཅིག་ཡིན། དེས་རྟག་བཞུགས་ Bridge Server བརྒྱུད་ནས་ལིན་ཧྲུའུ་འཕྲིན་བསྐུར་དོན་རྐྱེན་(消息事件)དེ་གཏན་འཁེལ་བའི་ CLI འཐུམ་རིམ་(进程)ལ་སྐྱེལ་བ་དང་། དེ་ནས་ CLI ཀྱི་ལན་དེ་ལིན་ཧྲུའུ་ཚོགས་པའམ་ལིན་ཧྲུའུ་སྒེར་གཏུག་(私聊)ནང་སླར་ཡང་བསྐུར་བ་ཡིན།

སྤྱོད་ཡུལ་གྱི་གནས་ཚུལ：
- ཚོགས་པའི་ནང་ལིན་ཧྲུའུ་ཚོགས་པ་བརྒྱུད་ནས་མཉམ་འབྲེལ་སྒོས་ AI ལས་བཟོའི་ལས་འགན་གཏོང་བ
- PC མེད་པའི་གནས་ཚུལ་(ལག་ཐོགས་ཁ་པར་དང་། སྒྲོམ་བུ)འོག ཐག་རིང་ནས་ཞབས་ཞུ་ཆས་སྟེང་གི་ Claude Code ཚོད་འཛིན་བྱེད་པ
- ཞབས་ཞུ་ཆས་གཅིག་གི་སྟེང་གི་ AI འཐུམ་རིམ་མང་པོ་དེ་ལིན་ཧྲུའུ་ཚོགས་པ་མི་འདྲ་བར་སྦྲེལ་ནས་རྣམ་གྲངས་ལོགས་སུ་དབྱེ་བ
- ཚུལ་མཐོང་(浏览器)ཡང་ན་ཨན་ཀྲོ་ཌ་端 བརྒྱུད་ནས་དངོས་ཡུལ་སུ་ AI ཐག་གཅོད་འཕར་ཚད་དང་། ཉིན་ཐོ། གླེང་མོལ་ Timeline ལྟ་བ

ལག་རྩལ་བསྒྲིགས་(技术栈)：
- **མདུན་ས་(前端)**：Vue 3 + Vite + TypeScript + Pinia + Tailwind + xterm.js
- **རྒྱབ་ས་(后端)**：Node.js 20+ + Express + ws + node-pty + ssh2
- **གཞི་གྲངས་མཛོད་(数据库)**：SQLite + Drizzle ORM
- **འཐུམ་རིམ་དོ་དམ**：tmux + PM2

```
bacs-bridge-server/
├── src/
│   ├── client/               # Vue 3 མདུན་ས
│   │   ├── views/            # ཤོག་གྲངས་9（Home/Bindings/Machines/Providers/Terminal/Logs/Settings/Login/Layout）

│   │   ├── composables/      # useAuth / useDeviceId / useTerminalSession
│   │   └── router/           # vue-router
│   ├── server/               # Express རྒྱབ་ས
│   │   ├── routes/           # auth/bindings/machines/providers/sessions/logs/timeline/settings/models/health
│   │   ├── channel/          # ལིན་ཧྲུའུ་ WS Channel + ཚོང་ལས་མཚོན་བྱེག་འཇུག་ཁ
│   │   ├── cli/              # CC / Codex adapter
│   │   ├── executor/         # ས་གནས་ + SSH ལག་བསྟར་ཆས་(执行器)
│   │   ├── terminal/         # pty-bridge + ws-server（Web Terminal）
│   │   ├── auth/             # JWT + TOTP + བློ་བཀལ་སྤྱོད་ཆས་མཛུབ་རིས་(信任设备指纹)
│   │   ├── db/               # Drizzle schema + གནས་སྤོ་ཡིག་ཆ་(迁移文件)
│   │   └── session/          # གླེང་མོལ་རྣམ་པའི་འཕྲུལ་ཆས་(idle / working / awaiting_choice)
│   └── shared/               # མདུན་རྒྱབ་མཉམ་སྤྱོད་རིགས་དབྱིབས་(类型)
├── scripts/                  # migrate-db / seed-admin / migrate-bindings
```

### སྦྲེལ་མཐུད་(绑定)གསར་བཟོ་བ

སྦྲེལ་མཐུད་(绑定)འདེམས་ཐོ་ནང་འཛུལ་→ གསར་སྣོན་：
- **སྦྲེལ་མཐུད་མིང་**：རང་འདོད་(དཔེར་ན་ `cc-projectA`)
- **འཕྲུལ་ཆས་(host)**：ས་གནས་ཡང་ན་སྔོན་སྦྱོང་བཟོས་པའི་ཐག་རིང་འཕྲུལ་ཆས་བདམས
- **CLI རིགས**：cc / codex
- **ཞབས་ཞུ་སྤྱོད་མཁན**：བཟོས་ཟིན་པའི་ཞབས་ཞུ་སྤྱོད་མཁན་བདམས
- **དཔེ་དབྱིབས་(模型)**：ཞབས་ཞུ་སྤྱོད་མཁན་ནས་རང་འགུལ་འཚོལ་བ། འཚོལ་མ་ཐུབ་ན་སོར་སྐྱེལ་དཔེ་དབྱིབས་བདམས་ཡང་ན་རང་འདོད་དཔེ་དབྱིབས་ ID ནང་འཇུག
- **Effort**：དཔེ་དབྱིབས་ཀྱི་ maxEffort གཞིར་བཟུང་གདམ་བྱའི་ཚད་གྲངས་སྟོན་པ
- **ལིན་ཧྲུའུ་བཀོལ་སྤྱོད་ཆ་བཞི**：App ID / Secret / Verification Token / Encrypt Key

ཉར་ཚགས་རྗེས། Bridge ཀྱིས་རང་འགུལ་གྱིས་ལིན་ཧྲུའུ་འབྲེལ་མཐུད་ཡུན་རིང་ WS ལ་སྦྲེལ་རྩིས་བྱེད། རྣམ་པ་ `online` ལ་གྱུར་རྗེས་ལིན་ཧྲུའུ་ཚོགས་པའི་ནང་ @ འཕྲུལ་མི་བེད་སྤྱོད་བྱེད་ཐུབ།

**bacs-android** ནི་མ་ལག་འདིའི་གཞུང་ཕྱོགས་ཨན་ཀྲོ་ཌ་端 App ཡིན། ཁྱེད་ལག་ཐོགས་ཁ་པར་སྟེང་：

- 🔔 དངོས་ཡུལ་དུ་ Bridge Timeline སྐྱེལ་བ་ལེན་པ། AI ཀྱི་ལན་ནི་ IM འཕྲིན་བསྐུར་དང་འདྲ་བར་འབྱོར་བ
- ⌨️ ཐད་ཀར་ལག་ཐོགས་ཁ་པར་སྟེང་བཀའ་བཏང་ནས་ Bridge ལ་བསྐྱེལ་བ། ལིན་ཧྲུའུ་ཁ་ཕྱེ་མི་དགོས
- 📊 སྦྲེལ་མཐུད་ཡོངས་ཀྱི་ CLI འཐུམ་རིམ་གྱི་རྣམ་པ་ལྟ་བ་(online / offline / awaiting_choice)
- 📜 ལོ་རྒྱུས་གླེང་མོལ་དང་མ་ལག་ཉིན་ཐོ་བལྟ་བ
- 🔐 TOTP ཐེངས་གཉིས་བདེན་དཔྱད་ནང་འཛུལ་+ སྤྱོད་ཆས་མཛུབ་རིས་བློ་བཀལ་(信任)
- 🌙 མ་ལག་གི་ Light / Dark བརྗེ་མདོག་དང་མཉམ་དུ་འགྲོ་བ

**འབྲེལ་མཐུད་ཐབས་ལམ**：App འགོ་བརྩམས་རྗེས་ Bridge Server ཁ་བྱང་ནང་འཇུག（དཔེར་ན་ `http://192.168.1.100:3456`），སྒྲིག་འགོད་ཤོག་ངོས་ཀྱི་མགྱོགས་ནང་འཛུལ་རྩ་གཉིས་ཨང་(二维码)བཤེར་ནས་མཐེབ་གཅིག་ནང་འཛུལ་བཟོ་ཐུབ་(སྐར་མ་60 གྱི་ཐུང་ཡུན་ JWT → ཁྱེར་མཁན་མཐའ་སྣེ་(客户端)ཡིས་ཡུན་རིང་ token exchange བྱེད་པའི་ཇུས་གཞི། རྩ་གཉིས་ཨང་ཤོར་བ་(泄露)སྔོན་འགོག)

**ཨན་ཀྲོ་ཌ་端 རྣམ་གྲངས་**ཀྱི་ཡིག་ཆ་ཆ་ཚང་ [bacs-android README](https://github.com/LengendXing/bacs-android#readme) ནང་གཟིགས།

## ❓ རྒྱུན་མཐོང་དཀའ་ངལ

**དྲི་བ：ལིན་ཧྲུའུ་འཕྲིན་བསྐུར་བསྐུར་རྗེས་ལན་མེད་པ？**
ལན：སྔོན་ལ་ཞིབ་བཤེར་བྱོས：① སྦྲེལ་མཐུད་རྣམ་པ་ `online` ཡིན་མིན；② ཞབས་ཞུ་ཆས་ཀྱི་ tmux session གསོན་ཡོད་མེད；③ ཞབས་ཞུ་སྤྱོད་མཁན་གྱི་ API Key ནུས་ལྡན་ཡིན་མིན；④ ཉིན་ཐོ་འདེམས་ཐོ་ནས་དངོས་ཡུལ་དུ་རྒྱབ་ས་ཉིན་ཐོ་ལྟ་བ

**དྲི་བ：ཐག་རིང་འཕྲུལ་ཆས་ནང་འཛུལ་མ་ཐུབ་(Not logged in)པ？**
ལན：v1.0.8 ནང་བཟོ་བཅོས་བྱས་ཟིན——`bash -ilc` ཀྱིས་དཀྲིས་པའི་ tmux འགོ་བརྩམས་བཀའ་བེད་སྤྱོད་བྱས་ནས་ཐག་རིང་གི་ rc ཡིག་ཆ་བོད་འཇུག་བྱེད་པར་ཁག་ཐེག་བྱོས། གལ་ཏེ་ད་དུང་དཀའ་ངལ་ཡོད་ན། ཐག་རིང་གི་ `~/.bashrc` ནང་ `claude` / `codex` ལག་བསྟར་ཆས་རྙེད་ཐུབ་མིན་ལྟ་བ

**དྲི་བ：SSH ལས་འགན་ཡུན་རིང་སྐབས་འབྲེལ་བ་ཆད་པ？**
ལན：v1.0.10 ནང་བཟོ་བཅོས་བྱས་ཟིན——སྙིང་བརྡ་(心跳)སྐར་མ་5 ནས་སྐར་ཆ་30 ལ་བསྒྱུར་བ། དེ་མིན་སྐར་ཆ་60 གྱི་རང་འདུན་སྟོང་གཡེང་འབྲེལ་བ་ཆད་པ་(空闲断连)བསུབས་ཡོད།

**དྲི་བ：tmux session མེད་ན་ཇི་ལྟར་བྱ་དགོས？**
ལན：Bridge ཀྱིས་སྦྲེལ་མཐུད་དྲ་ཐོག་(上线)སྐབས་རང་འགུལ་གྱིས་འཚམས་པའི་ session བཟོ་བ（དཔེར་ན་ `cc-xxx`、`codex-xxx`）。ལག་བསྟར་བྱེད་མི་དགོས།

**དྲི་བ：Web Terminal ཚུལ་མཐོང་སྒེའུ་ཁུང་ཁ་བརྒྱུད་ན་ལས་སྒོའི་འཐུམ་རིམ་གསོད་ཐུབ་བམ？**
ལན：**མི་ཐུབ**། Web Terminal གྱིས་ tmux མཐའ་སྣེ་(客户端)ཙམ་ལ་ detach བྱེད། ལས་སྒོའི་ session རྒྱུན་མཐུད་དུ་འཁོར་སྐྱོད་བྱེད། ཚིག་ག་(代码)རིམ་པར་བཙན་བཙུགས་འགོག་སྲུང་ཡོད：`session-name` དཀར་ཐོ་རྒྱུད་དུས་(白名单正则)འགྲོ་བ། ཁ་བརྒྱུད་པའི་ལམ་བུས་ `tmux kill-session` ལག་བསྟར་གཏན་ནས་བྱེད་མི་སྲིད།
