正常输出流  →  无需转发，静默处理

任务完成    →  发送结果摘要

claude --output-format stream-json -p "你的任务"

{"type":"result","subtype":"success","result":"..."}

关键优化点总结
问题解法长任务中途刷屏缓冲聚合，每 N 秒汇总一次输出决策点识别正则匹配 + 超时空闲检测双保险等待回复超时asyncio.wait_for 设 10 分钟，超时走保守策略多任务并发每个任务独立 PTY + Future，按 session_id 隔离进程意外退出监听 SIGCHLD，自动通知飞书
