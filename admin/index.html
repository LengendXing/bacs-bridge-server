📖 飞书自建应用配置指引
1
创建自建应用

打开 飞书开放平台 → 创建企业自建应用 → 填写应用名称和描述。

2
获取 App ID 和 App Secret

应用首页 → 凭证与基础信息 → 复制 App ID 和 App Secret，填写到下方绑定表单中。

3
添加机器人能力

左侧菜单 → 添加应用能力 → 添加「机器人」。

4
配置事件订阅（长连接）

左侧菜单 → 事件与回调 → 事件配置 → 选择 「使用长连接接收事件」（无需填写回调地址）→ 添加事件 im.message.receive_v1（接收消息）→ 保存。

5
配置权限

左侧菜单 → 权限管理 → 开启以下权限：
• im:message - 获取消息内容
• im:message:send - 发送消息
• im:message:send_as_bot - 以机器人身份发消息

6
发布应用

左侧菜单 → 版本管理与发布 → 创建新版本 → 发布 → 等待管理员审批通过。

7
完成绑定

在飞书内搜索你创建的应用 → 发送消息 → 消息通过长连接被本服务接收 → 自动路由到绑定的 CC 进程。

⚠️ 每次新增飞书应用都需要：新建应用 → 获取新的 App ID + App Secret → 在此面板绑定。在本页完成绑定后，消息会自动路由到对应 CC 进程。
